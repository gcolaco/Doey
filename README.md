# Doey

###### Doey é um aplicativo de lista de afazeres, o desafio desse aplicativo foi a utilização de persistencia de dados do usuário(Core Data), permitindo o usuário salvar dados no aplicativo ele manter esses dados salvos mesmo após o app ser encerrado.
